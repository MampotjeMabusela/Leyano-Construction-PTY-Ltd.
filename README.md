# Leyano Construction Website

Official marketing website for **Leyano Construction (PTY) LTD**.

This project is a responsive static website built with HTML, CSS, and vanilla JavaScript. It showcases the company brand, services, project portfolio, founder profile, and contact channels (including WhatsApp and social links).

## Live Purpose

- Present Leyano Construction professionally online
- Build trust with potential clients
- Showcase completed and in-progress projects
- Make it easy for clients to contact the business

## Features

- Fully responsive design for:
  - phones
  - tablets
  - laptops/desktops
  - large screens/TVs
- Branded UI theme aligned with Leyano logo
- Founder/CEO trust section
- Advanced project gallery:
  - featured project layout
  - category filters
  - lightbox image popup
- WhatsApp click-to-chat integration with prefilled message
- Social links (Facebook, Instagram, TikTok)
- Contact section with updated address and details

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- Font Awesome icons
- Google Fonts

## Run Locally

You can open `index.html` directly, or run a local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/index.html
```

## Deployment

This site is ready for deployment on platforms like **Vercel** via GitHub import.
